# DESIGN PORTFOILO

## 프로젝트 소개

```bush
본 저장소는 개인 디자인 포트폴리오 웹사이트를 관리하고 배포하기 위한 저장소입니다.
디자인 작업물과 웹 퍼블리싱 결과물을 정리하여 하나의 웹 포트폴리오로 구성하였습니다.
```

포트폴리오 URL : https://silverstar9482.github.io/PORTFOILO_DESIGN/

---

## 사용 목적

- 본 프로젝트는 개인 디자인 포트폴리오를 온라인으로 배포하기 위해 제작되었습니다.
- 외부 호스팅 서비스에 의존하지 않고 GitHub를 활용하여 안정적으로 포트폴리오를 관리·공개하는 것을 목표로 합니다.

## 사용 방법

- 상단에 안내된 배포 URL에 접속하여 디자인 포트폴리오를 확인할 수 있습니다.
- 별도의 설치나 실행 과정 없이 웹 브라우저에서 바로 이용 가능합니다.

## 기술 스택

- **HTML5 / CSS3** : 레이아웃, 스타일링 직접 구현
- **JavaScript** : 인터랙션 구현 및 구조 개선
- **GitHub** : 소스 코드 관리 및 배포
